# 📄 Document Extractor - Smart Document Analysis

An AI-powered web application that extracts names and phone numbers from images and PDF documents using OCR technology.

## ✨ Features

- 📸 **Image Processing** - Extract text from JPG, PNG images using Tesseract OCR
- 📑 **PDF Support** - Parse and extract text from PDF documents
- 🔍 **Smart Extraction** - Automatically identify names and phone numbers using regex patterns
- 💾 **Database Storage** - Store all extracted data in SQLite database
- 📊 **Document History** - View all previously processed documents
- 🎨 **Modern UI** - Beautiful, responsive interface with animations
- 🚀 **Real-time Processing** - Instant extraction with visual feedback

## 🛠️ Tech Stack

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **SQLite3** - Lightweight database
- **Tesseract.js** - OCR engine for images
- **pdf-parse** - PDF text extraction
- **Multer** - File upload handling

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling with animations
- **JavaScript (ES6+)** - Interactive features
- **Font Awesome** - Icons
- **Google Fonts** - Typography

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm (Node Package Manager)
- Modern web browser
